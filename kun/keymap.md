# Layers

## Usage

Can move to Every Layer with NumberPad (1 - 5)
Except _BASE_LAYER(default), Press Layer.

`_BASE_LAYER` = 0 layer (Default)
`_FN1_TOGGLE_LAYER` = movement layer

    /*
	* Layer _BASE_LAYER
